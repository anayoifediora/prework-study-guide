# Prework Study Guide Webpage

## Description

This project is a Prework Study guide that will enable the Bootcamp students study the various topics such as HTML, CSS, Git and Javascript. The website also helps the student to pick the courses randomly.

## Usage
Each section contains notes about a particular topic such as HTML, CSS, JavaScript and Git. If the ChromeDevTools is opened, students can see the JavaScript running. It lists the topic learned and also randomly picks a topic for the student to read first.
