CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • "rel" specifies the relationship between the current document and the external document.
  • "href" specifies the location or URL of the external source.
  • A universal CSS selector is denoted by "*"
  • A class CSS selector is denoted by "." followed by the attribute

JavaScript

  • A variable is a named container that alows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Data types include Strings, Number, Boolean, Array and Object
  • Functions can be used to categorize or group a series of code you wish to re-use. It executes when you call the function name in the code.
  • The "For" loop uses a predictable pattern of indices to perform a task on all items on an array, using a single code block executed over and over.
  • The "For" loop contains 3 important statements: the starting point, the condition and the final statement that allows the arrar to iterate over each item.
